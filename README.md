# react-native-ux-cam

## Installation
`$yarn add file:/path-to-the-uxcam-react-wrapper`
`$react-native link react-native-ux-cam`

## Usage
```javascript
import RNUxcam from 'react-native-ux-cam';


RNUxcam.startWithKey('YOUR API KEY');
```
# For testing example app
